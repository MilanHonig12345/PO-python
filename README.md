# PO-python door Mayla Hinze en Milan Honig

**onderwerp:**
Wij hebben gekozen om een spelletje te maken waar je moet overleven in een bunker. Van te voren moet je spullen verzamelen die nodig kunt hebben in de bunker. In de bunker moet je de juiste keuzes maken om te overleven. Je begint met een hoeveelheid levens en die gaan er van af door verkeerde keuzes. Je hebt ook de keuze om naar buiten te gaan op zoek naar spullen, maar je hebt het risico om aangevallen te worden door een zombie en daarbij dus een klein extra spelletje moet spelen om te overleven.
