# PO-python door Mayla Hinze en Milan Honig

**onderwerp:**
Wij hebben gekozen om een spelletje te maken waar je tegen een zombie moet vechten door verschillende spellen tegen de zombie te spelen. Jij hebt 4 levens. De zombie heeft 4 levens. Elke ronde speel je een mini-game. Win jij? Dan verliest de zombie 1 leven. Verlies je? Dan verlies jij 1 leven. Wie als eerste op 0 staat, verliest.
